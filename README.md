# Krisz Writes

Personal writing space for stories, memories, and chronicles.

## Structure

- **`_posts/`** - Individual stories and memories (blog format)
- **`_books/`** - Long-form narratives organized by chapters
- **`JACOB_SYSTEM_PROMPT.md`** - Context for AI assistant

## Writing on Mobile

You can write directly on GitHub mobile:

1. Go to https://github.com/kmcconnell/krisz-writes
2. Navigate to `_posts/` or `_books/`
3. Tap "Add file" â†’ "Create new file"
4. Name it with date format: `YYYY-MM-DD-title.md`
5. Write your content in Markdown
6. Commit directly to main
